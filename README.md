# MyCalendar

iOS 16.1+ 기기의 잠금화면과 Dynamic Island에서 동작하는 캘린더 위젯입니다.

## 주요 기능

- 잠금화면에서 월간 달력 표시
- Dynamic Island 지원
- 다크모드/라이트모드 지원
- 주말 색상 구분 (토요일: 파란색, 일요일: 빨간색)
- 오늘 날짜 하이라이트

## 요구사항

- iOS 16.1 이상
- Xcode 15.0 이상

## 설치 방법

1. 프로젝트를 클론합니다
2. Xcode에서 프로젝트를 엽니다
3. 실행할 기기를 선택하고 빌드합니다
